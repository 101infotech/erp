## Color Coding

The UI uses consistent color coding:

-   **Lime/Green** (`lime-400/500`) - Primary actions, active states
-   **Blue** (`blue-400/500`) - Information, secondary actions
-   **Yellow** (`yellow-400/500`) - Password-related actions
-   **Red** (`red-400/500`) - Delete/destructive actions
-   **Slate** (`slate-700/800`) - Background, borders, inactive states
